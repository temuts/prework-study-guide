Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Parent elements contain child elements
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users
  • There is always a starting tag and closing tag. Closing tag begins with a forward slash
  • First declaration tells the browser what document type should be expected

CSS

    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • A selector called .card will implement separation between points
